# Install

* git clone https://github.com/hamti2018/mpPoolInfoBot
* cd mpPoolInfoBot
* npm install
* sudo mv .env-example .env
* mv config-example.js config.js
* sudo nano .env
* sudo nano config.js

Отредактируйте файл .env и config.js

# Start

* npm start

# Start use pm2

* npm install pm2 -g
* pm2 start index.js
